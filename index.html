Web & HTML Notes

Internet:

HTML stands for "HyperText Markup Language"
Hyper Links - Typical internet links
HTML documents form the majority of the content on the web. HTML documents contain text content which describes "what you see" and markup which describes "how it looks".

Major Pieces of the Internet:

You/User
Computer/Browser
Internet uses HTTP
Servers

HTML

HTML is the heart of the web.
HTML is made up of:
- Text content (what you see)
- Markup (what it looks like)
- References to other documents (i.g. images & video)
- Links to other pages

HTML Markup

"b" tag - bolds the contents between the opening and ending tag
"em" tag - similarly, the "em" tag italicizes the contents between tages

Note: Omitting closing tag will apply anything after the opening tag i.e. everything after opening bold or italicized.

HTML Attributes

HTML Attributes are used within tags to include links or images.
For example, the Anchor tag used with "href" attribute creates a hyperlink.
Like this: Google.com

Images

Another usful element is for inlcuding images. img src="URL" alt=text
--> Note: Generally alt=text is 'required'/best practice so people know what they should be viewing if an image is expected.

Whitespace:
br tag = break (void tag)
p tag = paragraph

Why have ""br" and "p - /p" tags?
br - the break tag is "inline" whereas
p - paragraph tag is a "block"

The key distinction between inline and block elements is that block elements create ‘invisible boxes’ around the contents

Container Elements

Container elements: elements that do nothing but contain stuff inside themselves

Two Types: Span / Div
span - /span - inline container
div - /div - block container div class=“bar”> text /div
